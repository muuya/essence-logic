## 📝 使用方式更新

### 运行脚本

**之前：**
```bash
./dev.sh
python deploy.py
```

**现在：**
```bash
./scripts/dev.sh
python scripts/deploy.py
```

### 配置文件

**之前：**
```bash
python deploy.py --config deploy.config.json
```

**现在：**
```bash
python scripts/deploy.py --config config/deploy.config.json
# 或（自动查找）
python scripts/deploy.py --config deploy.config.json
```
